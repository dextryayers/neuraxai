## ğŸš€ Panduan Instalasi Cepat

Ikuti langkah-langkah ini untuk menjalankan NeURAX AI di mesin lokal Anda:

### 1. Persiapan Awal
Pastikan Anda telah menginstal:
*   **Node.js** (Versi 18+)
*   **npm** atau **yarn**

### 2. Clone Repository
```bash
git clone https://github.com/username-anda/neurax-ai.git
cd neurax-ai
```

### 3. Instalasi Dependensi
```bash
npm install
```

### 4. Konfigurasi API Key
Buat file `.env` di root folder proyek dan tambahkan API Key Gemini Anda:
```env
# Dapatkan API Key gratis di aistudio.google.com
API_KEY=masukkan_kunci_api_anda_disini
```

### 5. Jalankan Aplikasi
```bash
npm run dev
```
Buka browser dan akses: `http://localhost:5173`
